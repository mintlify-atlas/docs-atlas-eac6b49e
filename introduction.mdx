Sheets is a terminal UI (TUI) application that lets you view and edit spreadsheet data directly in your terminal. It reads and writes standard CSV and TSV files, supports built-in formulas, and can be driven entirely from the command line without opening the interactive editor.

## Two ways to use Sheets

**Interactive TUI** — launch with a file path (or no arguments) to open the full-screen editor:

```bash
sheets budget.csv
```

**Headless CLI** — pass a file and one or more cell references or assignments to query or modify data without opening the editor:

```bash
# Read a cell
sheets budget.csv B9

# Set two cells
sheets budget.csv B7=10 B8=20
```

The CLI mode writes changes back to the file and exits immediately, making it easy to script against CSV data.
