## Get started in seconds

<Steps>
  <Step title="Install Sheets">
    ```bash
    go install github.com/maaslalani/sheets@main
    ```
  </Step>
  <Step title="Open a CSV file">
    ```bash
    sheets budget.csv
    ```
  </Step>
  <Step title="Navigate with vim keys">
    Use `h`, `j`, `k`, `l` to move between cells. Press `i` to start editing a cell. Press `ESC` to return to normal mode.
  </Step>
  <Step title="Save and quit">
    Press `:` to open the command prompt, then type `wq` and press `Enter` to save and exit.
  </Step>
</Steps>

<Note>
  Sheets supports both CSV and TSV files. The file format is detected automatically from the file extension (`.csv` or `.tsv`).
</Note>
