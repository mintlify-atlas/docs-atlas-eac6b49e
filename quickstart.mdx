This guide walks you through installing Sheets, opening a CSV file, making edits, and saving — from scratch.

<Steps>
  <Step title="Install Sheets">
    If you have Go installed, run:

    ```bash
    go install github.com/maaslalani/sheets@main
    ```

    For other installation options, see the [Installation](/installation) page.
  </Step>
  <Step title="Open a CSV file">
    Pass a file path to open it in the interactive editor. If the file does not exist, Sheets creates it when you save:

    ```bash
    sheets budget.csv
    ```

    You can also pipe CSV data directly from stdin:

    ```bash
    sheets <<< "ID,Name,Age
    1,Alice,24
    2,Bob,32
    3,Charlie,26"
    ```

    <Tip>
      Sheets also accepts piped input from other commands: `cat budget.csv | sheets`
    </Tip>
  </Step>
  <Step title="Navigate the spreadsheet">
    Use vim-style keys to move between cells:

    | Key | Action |
    |-----|--------|
    | `h` / `l` | Move left / right |
    | `j` / `k` | Move down / up |
    | `gg` | Jump to the first row |
    | `G` | Jump to the last row |
    | `0` | Jump to the first column |
    | `$` | Jump to the last non-empty column |

    <Tip>
      If you use Vim, most movement keybindings will feel familiar. Sheets follows Vim conventions throughout.
    </Tip>
  </Step>
  <Step title="Edit a cell">
    Move to any cell, then press `i` to start editing. Type your value, then press:

    - `Enter` to commit and move down
    - `Tab` to commit and move right
    - `ESC` to leave insert mode without committing

    To clear a cell and start editing immediately, press `c`.
  </Step>
  <Step title="Add a formula">
    Move to an empty cell and press `i` to enter insert mode. Type a formula starting with `=`:

    ```
    =SUM(B1:B3)
    ```

    Press `Enter` to commit. The cell displays the computed result. Supported functions are `SUM`, `AVG`, `MIN`, `MAX`, and `COUNT`.
  </Step>
  <Step title="Save the file">
    Press `:` to open the command prompt, then type `w` and press `Enter`:

    ```
    :w
    ```

    To save to a different path:

    ```
    :w path/to/output.csv
    ```
  </Step>
  <Step title="Quit">
    Press `:` to open the command prompt, then type `q` and press `Enter`:

    ```
    :q
    ```

    To save and quit in one step:

    ```
    :wq
    ```
  </Step>
</Steps>

## CLI usage reference

Sheets can be used without the TUI for scripting:

```
USAGE
  sheets [file.csv]
      Launch sheets interactively.

  sheets [file.csv] [cell|range|cell=value|range=value]...
      Query or modify a cell through the command-line interface.

  sheets < input.csv
      Launch sheets interactively and load CSV through stdin.
```

Read a single cell:

```bash
sheets budget.csv B9
```

Read a range:

```bash
sheets budget.csv B1:B3
```

Set cells without opening the editor:

```bash
sheets budget.csv B7=10 B8=20
```
